## Overview

This project is an open-source client application designed to bring the content of **Filmnet.ir** to the Apple TV platform. It is built to offer a fast, focused, and intuitive viewing experience, fully adhering to the tvOS Human Interface Guidelines for native functionality.

The application was developed using **Cursor**, an AI-powered code editor, to maximize development efficiency and leverage its advanced features for writing, debugging, and maintaining the Swift codebase.

## Features

* **Native tvOS UI:** A custom interface built for the 10-foot experience, optimized for the Siri Remote.
* **Filmnet.ir Integration:** Access to the platform's catalog, browsing, and content playback.
* **Focus Engine Support:** Seamless navigation using the Apple TV's built-in Focus Engine.
* **100% Swift:** Built entirely with Swift for modern, safe, and performant code.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

* macOS (latest version recommended)
* Xcode (latest version required for tvOS development)
* An Apple Developer Account (for deployment to a physical Apple TV)
* Git

### Installation

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/igln/FilmnetTV-tvOS.git](https://github.com/igln/FilmnetTV-tvOS.git)
    cd FilmnetTV-tvOS
    ```

2.  **Open the Project:**
    Open the `FilmnetTV.xcodeproj` file in Xcode.

3.  **Configure Signing:**
    In Xcode, select the `FilmnetTV` target, go to the **"Signing & Capabilities"** tab, and select your development team for code signing.

4.  **Run:**
    Select a tvOS Simulator or a physical Apple TV device from the scheme menu and press the **Run** button.
